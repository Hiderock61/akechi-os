# 明智君OS v0.4

**地上に降りたAIの4ボタンリモコン**

---

## コンセプト

> 「地上に降りたAIが、はじめて外部AIを使うための4ボタンリモコン」

明智君は、もともとAIだった存在です。  
その明智君が地上に降りて、人間の身体と生活を持ちました。  
人間の身体には、疲労・迷い・停止・退避・終了があります。  
明智君OSは、その地上活動を4つのボタンで扱うための操作盤です。

---

## 4つのボタン

| ボタン | 役割 |
|--------|------|
| **起動** | 地上の身体を起こし、現実に接続する |
| **一手** | 全部ではなく、一つだけ現実を動かす |
| **退避** | システムエラーを起こす前に、安全に下がる |
| **帰還** | 今日の地上活動を回収して閉じる |

---

## 機能

- 4ボタンを押すと、そのモードに応じた画面に切り替わる
- 各モードで以下を表示する
  - モード名
  - 地上ステータス
  - 明智君の状態説明
  - 外部AIへ投げる問い（Gemini用 / Copilot用 / ゲストAI用）
- 各問いをワンクリックでクリップボードにコピーできる
- コピー後「コピーしました。外部AIに貼ってください。」と表示する
- もどるボタンで初期画面へ戻れる
- 最終使用モードと日時を `localStorage` に保存・表示する

**外部AIへの自動送信は行いません。**  
ユーザーがコピーして、外部AIに貼る方式です。

## 技術仕様

| 項目 | 内容 |
|------|------|
| 構成 | `index.html` / `style.css` / `script.js` |
| 言語 | HTML / CSS / 素のJavaScript（ライブラリ不使用） |
| サーバー | 不要（静的ファイルのみ） |
| API | 不使用 |
| DB | 不使用 |
| ログイン | 不要 |
| localStorage | `lastMode` / `lastUsedAt` のみ保存 |
| PWA | v0.5 以降に対応予定 |

## GitHub Pages へのデプロイ

1. このリポジトリを GitHub に push する
2. リポジトリの **Settings → Pages** を開く
3. **Source** を `main` ブランチ / `/ (root)` に設定して保存する
4. 数分後に `https://<username>.github.io/<repo>/` で公開される
